Список переменных, которые вы можете использовать в тексте сообщения. Например, если вы используете ${user_id}, оно будет заменено на Telegram ID пользователя, ${username} будет заменено на имя пользователя, ${user_message_text} будет заменено на текст сообщения пользователя, а ${web_api}:{ссылка} будет заменено на ответ с Web-API.

Текстовое поле, где вы должны ввести название команды для вашего Telegram бота. Например: "start".

Текстовое поле, где вы можете ввести саму команду, которую telegram бот будет распознавать. Например, если вы ввели "start" в поле "Название команды", то здесь вы можете ввести "/start" или любой другой текст, который telegram бот будет интерпретировать как команду.

Текстовое поле, где вы можете ввести CallBack текста, который будет отправлен Telegram боту при нажатии на кнопку встроенной клавиатуры.

Текстовое поле, где вы должны ввести текст сообщения, которое Telegram бот будет отправлять пользователю после обработки команды.
